Llamada a una función

```
CALL DIR
```
siendo `DIR` una dirección de memoria.

Una llamada a una función mezcla diferentes conceptos:
* Uso de la pila para almacenar la dirección de retorno.
* Salto a una dirección de memoria.

#### Debe de ir acompañado de la instrucción [`RET`](/es/instructions/RET).
