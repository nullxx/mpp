Cargar los flags con el valor del acumulador.

```
FLAGS <= AC
```

### Ejemplo

Imaginamos que el acumulador tiene el valor de 0x81 que en binario se representa como 10000001.
Al ejecutar la operación `SFA`: 
* El flag ZERO obtendrá el valor del bit más representativo del acumulador.
* El flag CARRY obtendrá el valor del bit menos representativo del acumulador.

```
10000001
^      ^
|      |
FC     FZ
```


### Comando

```
SFA
```
