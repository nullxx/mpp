OR-EXCLUSIVA (XOR) entre el valor de un registro y el del acumulador.

**El resultado se guarda en el acumulador**.

### OR-EXCLUSIVA de un valor inmediato y el acumulador

```
XOR 0F
```

### OR-EXCLUSIVA de un registro de propósito general y el acumulador

```
XOR RB
```

### OR-EXCLUSIVA del acumulador y el acumulador

```
XOR AC
```
