Cargar valor de memoria en el acumulador.

A diferencia de la operación [`LDA`](LDA), ésta no espera un valor inmediato, ya que coge la dirección de memoria de los registros RD-RE.

**El resultado se ve reflejado el acumulador**

```
AC = READMEM(RD-RE)
```


### Comando

```
LDAX
```
