Suma lógica (OR) entre el valor de un registro y el del acumulador.

**El resultado se guarda en el acumulador**.

### Suma lógica de un valor inmediato y el acumulador

```
OR 0F
```

### Suma lógica de un registro de propósito general y el acumulador

```
OR RB
```

### Suma lógica del acumulador y el acumulador

```
OR AC
```
