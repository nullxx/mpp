Guardar el valor del acumulador en memoria.

A diferencia de la operación [`STA`](STA), ésta no espera un valor inmediato, ya que coge la dirección de memoria de los registros RD-RE.

```
WRITEMEM(RD-RE)
```

### Comando

```
STAX
```

## Comprueba tus conocimientos

Crea un programa que guarde el valor 0xFF en la dirección de memoria 0x1324.

**Nota**: debes conocer y utilizar la operación: [`MOV`](MOV)
