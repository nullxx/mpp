Incrementar el valor de un registro

```
REGISTRO = REGISTRO + 1
```
Siendo REGISTRO el registro cuyo valor tiene que incrementar

### Incrementar el valor de un registro de propósito general

```
INC RB
```

### Incrementar el valor del acumulador.

```
INC AC
```
