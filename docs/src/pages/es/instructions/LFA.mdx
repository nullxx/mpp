Guardar el valor de los flags en el acumulador.

```
AC <= FLAGS
```

### Ejemplo

Imaginamos la siguiente situación:

- Flag ZERO = 1
- Flag CARRY = 1

Al ejecutarse la operación `LFA` tendremos en el acumulador el valor de 0x81.

```
10000001
^      ^
|      |
FC     FZ
```

### Comando

```
LFA
```
